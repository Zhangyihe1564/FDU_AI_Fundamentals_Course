# FDU_AI_Fundamentals_Course
这个项目是复旦大学人工智能基础课程（AIB210002）两次大作业的代码仓库。

我怀疑他们从来没有更改过大作业的题目。所以理论上，这个仓库的代码应该可以帮助未来的同学完成大作业。
但是别抄，我已经交过了。后来者可以用我已经写好的代码运行、调整来完成自己的作业。
实验报告参考文件我也放进来了。

这整个仓库的初始文件从课程指导创建的阿里云DSW实例上取得，原始文件包括两个.ipynb文件和一些数据集文件。

当然，第二个大作业的前两个TODO的代码是我自己写的，和原始文件无关。

本项目使用uv进行项目管理，对模型进行了GPU适配（仅RTX 50系，CUDA 13.0），如果需要修改请修改HW1/HW2文件夹下的pyproject.toml文件。
具体怎么修改可以上网搜索或者来PCS。（uv的使用也可以）。

对于使用，你需要在两个作业文件夹下分别使用uv sync来创建环境并在ide（如vscode，pycharm）中选择对应虚拟环境的python解释器。

第一个大作业(HW1)使用python 3.14，第二个大作业(HW2)使用python 3.13。

你既可以将代码clone到本地运行，也可以将仓库clone到阿里云DSW实例上运行（推荐使用GPU实例）。
相关的git使用方法请自行上网搜索。
